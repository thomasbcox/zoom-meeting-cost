Security overview

Effective date: 1 June 2026 · Meeting Cost Meter

Meeting Cost Meter is built to handle the little data it touches carefully — encrypted, minimal, and never shared. This page summarizes how the app is developed and operated.

Secure development

  • Every change ships through a reviewed pull request with an independent code review and a passing automated test suite before it can merge.
  • The code is scanned on every change by GitHub CodeQL (static application security testing).
  • Dependencies are kept current and monitored for known vulnerabilities (Dependabot + dependency alerts).
  • A pre-commit secret-scanning check prevents credentials from ever entering the source. Secrets are configured as environment variables, never committed.
  • The release branch is protected: changes merge only via reviewed pull request with automated checks passing.

Encryption & data handling

  • All traffic is served over HTTPS/TLS (HSTS enforced).
  • The presenter's configuration is stored AES-256-GCM encrypted at rest, with a per-user key tied to their Zoom account; plaintext never touches disk.
  • The live on-camera total is an aggregate computed inside the Zoom client and is never sent to our server.
  • Responses set Content-Security-Policy, nosniff, and no-store cache headers.

Data, retention & privacy

We hold only the presenter's own opportunity-cost estimates and the Zoom-provided context needed to run — never wages or salaries, and no HR, payroll, SSO, or directory data. Stored configuration is kept (encrypted) until you delete it or ask us to. For the full details, see our Privacy Policy.

Transparency

We do not currently hold SOC 2 or ISO 27001 certification, and we do not perform third-party penetration testing. We describe only the controls that are actually in place, and we will update this page if that changes.

Reporting a vulnerability

Found a security issue? Please email thomas+mcsupport@txl-lab.com with the subject SECURITY, rather than posting publicly. We aim to acknowledge within two business days and will coordinate a fix with you.
